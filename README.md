
## Package Dependency 

Run 'npm install' for a package dependency.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The app will automatically reload if you change any of the source files.

## Fake Backend

Run `npm run api` for a api server. The server will be up and running in `http://localhost:3000/`.
